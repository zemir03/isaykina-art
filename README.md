# Сайт-визитка художницы (локально)

## Как открыть
1. Открой папку `site/`
2. Дважды кликни по файлу `index.html`

Сайт откроется в браузере и будет работать без интернета (кроме шрифта Google Fonts).

## Заявки (отправка в Telegram через Cloudflare Worker)
Форма отправляет заявку через прокси (Cloudflare Worker) → Telegram-канал.
Токен бота хранится только на сервере Cloudflare, не в коде сайта.

### Настройка (один раз)
1. Зарегистрируйся на https://dash.cloudflare.com (бесплатно)
2. Перейди в **Workers & Pages → Create → Create Worker**
3. Удали код-пример, вставь содержимое файла `cloudflare-worker.js`
4. Нажми **Deploy**
5. Скопируй URL воркера (вида `https://имя.username.workers.dev`)
6. Открой `assets/app.js`, найди `const WORKER_URL = "YOUR_WORKER_URL";`
7. Вставь туда скопированный URL

После этого заявки будут приходить в Telegram-канал `@my_art_orders`.

## Хостинг Beget

Файлы на сервер заливай через **файловый менеджер** или **FTP/SFTP** после настройки доступа в панели.  
**Beget.API хостинга не загружает файлы сайта** — только управляет услугами (домены, DNS, FTP-аккаунты и т.д.). Кратко см. `BEGET_API.md`.
