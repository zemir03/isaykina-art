# Сайт‑визитка художницы (локально)

## Как открыть
1. Открой папку `site/`
2. Дважды кликни по файлу `index.html`

Сайт откроется в браузере и будет работать без интернета (кроме шрифта Google Fonts).

## Заявки (пока без отправки)
Форма сохраняет заявку **на этом ПК** и показывает окно «Спасибо», где можно нажать **«Скопировать»** и вручную отправить текст в Telegram.

Позже можно подключить автоматическую отправку в Telegram (ботом) — структура уже подготовлена в `assets/app.js` (функция `submitLead()`).
